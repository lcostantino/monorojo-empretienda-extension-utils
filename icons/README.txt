# Extension Icons

Icons generated from MonoRojo Funshirt logo (rm.png)

## Files:
- **rm.png** - Original logo source (400x400)
- **icon16.png** - Toolbar icon (16x16)
- **icon48.png** - Extension management page (48x48) 
- **icon128.png** - Chrome Web Store listing (128x128)

## Generation:
Icons were generated using ImageMagick:
```bash
magick icons/rm.png -resize 16x16 icons/icon16.png
magick icons/rm.png -resize 48x48 icons/icon48.png
magick icons/rm.png -resize 128x128 icons/icon128.png
```

All icons maintain the MonoRojo Funshirt branding.
